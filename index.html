Para los que ven el curso después del 13 de agosto del 2021 Para crear un Token de GitHub entrar a: settings > developer settings > personal access tokens > generate new token. Si solo lo quieres usar para tu repositorio marcar la opción “repo” agrégale una nota y selecciona un tiempo de expiración. Copia el token y guárdalo en un lugar seguro, ya que después de cerrar la página no podrás verlo de nuevo. Ahora puedes usar ese token cuando te piden la contraseña al momento de hacer push
